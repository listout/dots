## Key bindings

Super is modkey

| Keybind             | Action                          |
| ------------------- | ------------------------------- |
| `super + Return`    | Terminal                        |
| `super + space`     | rofi                            |
| `super + f`         | thunar                          |
| `super + {1-9,0}`   | focus on the `0-9` desktop      |

*any many more*
